# ✦ NoteSpace — Personal Notepad

Aplikasi notepad personal berbasis web. Berjalan langsung di browser, 
tanpa backend, tanpa database, 100% offline.

## Fitur
- Buat, edit, hapus catatan
- Auto-save otomatis
- Pencarian real-time
- Dark mode
- Penyimpanan localStorage

## Cara Pakai
Cukup buka `index.html` di browser.
